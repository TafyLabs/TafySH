🔐 SECURITY MODEL (3 Layer Defense)

Layer 1: Risk Classification
  • Pattern matching (regex) for dangerous commands
  • 4 levels: SAFE, MEDIUM, HIGH, CRITICAL

Layer 2: RBAC (Role-Based Access Control)
  • VIEWER (read-only)
  • OPERATOR (execute safe/medium)
  • ADMIN (approve/deny, manage)

Layer 3: Human-in-the-Loop Approval
  • Agent proposes
  • System classifies risk
  • User approves/denies
  • Action logged in audit trail
